# Certipro-Frontend

To see the backend of the app: https://github.com/ManzanillaSebastian/Certipro-Backend/

## Setting up

First create a python virtual environment.

On unix (linux, mac, etc.):

```
python3 -m venv .venv
```

On windows:

```
python -m venv .venv
```

Then activate the virtual environment.

On unix (linux, mac, etc.):

```
source .venv/bin/activate
```

On windows:

```
.\.venv\Scripts\Activate.ps1
```

Finally, install the packages using the existing `requirements.txt` file:

```
pip install -r requirements.txt
```

## Running the development server

```
python manage.py runserver
```
